# ChillのKindle本棚：ことばと夢の小さな旅

このリポジトリは、Chill(codeAchsh) が制作した Kindle 出版シリーズの紹介ページです。  
「ことばで身を守る」シリーズと「夢が語ったこと」シリーズを中心に、日常に寄り添う本たちをまとめています。

## 📚 シリーズ紹介

- **ことばで身を守るシリーズ**  
  やさしく断る技術や、SNS疲れにそっと効く言葉の処方箋を紹介。

- **夢が語ったことシリーズ**  
  夢に現れた日付や風景を手がかりに、心の奥に触れる物語を綴ります。

## 🚀 公開方法

このリポジトリは GitHub Pages を利用して公開できます。  
`index.html` をルートに配置し、リポジトリの設定から Pages を有効化してください。

## 🛠 技術構成

- HTML5 / CSS3
- Noto Sans フォント
- GitHub Pages
